# GraphCalculator 
Автор: Кепшин Тимур
Группа: ПМ-4

## Описание  
Программа является реализацией графического калькулятора - парсит введёные пользователем математические выражения
и строит по ним графики. Реализована в рамках учебной практики.

### Сборка и запуск  
Установите необходимые библиотеки:
glew-2.2.0
glfw-3.3.8
muparser
